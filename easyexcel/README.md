# easyexcel

## pom.xml依赖配置

    <!-- easyexcel -->
    <dependency>
        <groupId>com.alibaba</groupId>
        <artifactId>easyexcel</artifactId>
        <version>2.1.4</version>
    </dependency>

## 依赖的jar包
- 1.1 easyexcel-2.1.4.jar
- 1.2 asm-4.2.jar
- 1.3 cglib-3.1.jar
- 1.4 ehcache-3.2.3.jar
