# apache poi

## pom.xml依赖配置

    <!-- apache poi -->
    <dependency>
        <groupId>org.apache.poi</groupId>
        <artifactId>poi</artifactId>
        <version>3.17</version>
    </dependency>
    <dependency>
        <groupId>org.apache.poi</groupId>
        <artifactId>poi-ooxml</artifactId>
        <version>3.17</version>
    </dependency>

## 依赖的jar包
>apache poi
- poi-3.17.jar
- poi-ooxml-3.17.jar
